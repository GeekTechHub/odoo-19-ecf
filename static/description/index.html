Facturación Electrónica DGII - e-CF

República Dominicana

Módulo oficial para la generación, firma y envío de Comprobantes Fiscales Electrónicos (e-CF) según las especificaciones de la DGII (Dirección General de Impuestos Internos) de la República Dominicana.

  • Generación de XML e-CF (e31, e32, e33, e34, e41, e43–e47)
  • Firma digital con certificado PKCS#12 (.p12)
  • Envío automático a la API REST de la DGII
  • Seguimiento de estado con Track ID
  • Generación de código QR del comprobante
  • Entornos de Pruebas (TesteCF) y Producción
